REQUIREMENTS
    Users can complete multiple modules over time, but only one module at a time
    Challenge equals to Sub-Badge
    Challenges award XP points to the user profile
    A challenge may span multiple weeks
    Each challenge maps to 2-3 skills
    Main badges (e.g. teamwork, personal development) are persistent across modules and games
    Challenges contribute XP to main badges, regardless of game
    Existing legacy points must be supported and retained
    Each user (Players) has a profile showing modules, challenges, XP, skills, and badges
    Group = Team (terms are synonymous)
    Users (Players) are mapped to one group (team)
    Groups (teams) are divided by age ranges, with overlap allowed
    Users must not be placed in groups with significant age gaps (e.g. 10-year-old with 16-year-old)
    Modules can be game-specific (Minecraft, Rocket League, Fortnite)
    Badge progression is not tied to a specific game
    Center Admins should be able to create/update/delete modules and challenges(badges/sub-badges) in the app
    Users progress through Bronze → Silver → Gold → Platinum → Diamond
    The scale for levels currently:
            0 - 30 points: Bronze
            31 - 70 points: Silver
            71 - 120: Gold
            120 +: Platinum
    XP comes from completing challenges (badges/sub-badges) within modules
    Users (Players) should be able to submit evidence (e.g. screenshots), evidence is reviewed and approved by a coach or centre admin
    Dashboard looks the same across all groups
    Players can see current module, challenges, XP, badges
    Players cannot start new module before finishing existing one
    Web applicatin must be responsive and can run on mobile phones, tablets and desktop devices properly.
    Accessibility must be activeted for all pages.
